Este proyecto forma parte del Reto #2 de Alura, donde se analiza el comportamiento de los clientes de la empresa ficticia Telecom X. El objetivo principal es identificar factores críticos que influyen en la deserción de clientes (Churn) y proponer estrategias de retención basadas en datos.

Descripción del Proyecto
El proyecto utiliza un conjunto de datos en formato JSON que contiene información detallada de 7,267 clientes. A través de un análisis exploratorio y procesamiento de datos con Python, se busca entender por qué los clientes abandonan el servicio y qué segmentos son los más vulnerables.

Tecnologías Utilizadas
Python 3.x

Pandas: Para la manipulación y normalización de datos JSON.

NumPy: Para operaciones numéricas.

Matplotlib & Seaborn: Para la visualización de datos.

Google Colab: Entorno de desarrollo.
